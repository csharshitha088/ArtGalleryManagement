# Art Gallery Management System

A web-based Art Gallery Management System developed using PHP and MySQL to simplify the management of artworks, artists, exhibitions, customer bookings, and payments.

## Features

### Admin
- Secure Admin Login
- Manage Artworks
- Manage Exhibition Events
- View Customer Bookings
- Manage Customer Details
- View Payments
- Manage Gallery Information

### Customer
- User Registration & Login
- Browse Artworks
- Book Exhibition Events
- Make Payments
- Cancel Bookings
- Submit Feedback

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## ⚙️ Installation

1. Install XAMPP or WAMP.
2. Clone this repository.

```
git clone https://github.com/csharshitha088/ArtGalleryManagement.git
```

3. Move the project folder to the `htdocs` directory.
4. Start **Apache** and **MySQL**.
5. Open **phpMyAdmin**.
6. Create a database.
7. Import the provided `artwork.sql` file.
8. Open your browser and visit:

```
http://localhost/artgalleryproject
```
